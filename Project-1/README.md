# Number Guessing Game (C Language)

This is a simple number guessing game made using **C language**.  
I created this project while learning C programming.

The program generates a random number between **1 and 100**, and the user has to guess it.

---

## How the Program Works

- The computer selects a random number between 1 and 100
- The user enters a number as a guess
- The program gives hints:
  - "Little Higher" if the guess is smaller
  - "Little Lower" if the guess is larger
- Numbers outside the range 1–100 are rejected
- Only valid guesses are counted
- The game ends when the correct number is guessed

---

## Concepts Used

- Variables and data types
- `if-else` conditions
- `do-while` loop
- Random number generation (`rand()` and `srand()`)
- Input validation
- Basic C libraries
